# 身份：我是谁？

**QuickPaths** — 桌面悬浮路径快复制工具 + Claude Code 启动器。

在桌面显示一个悬浮呼吸光点，点击展开常用项目路径列表，一键复制或直接启动 `claude` 终端。内置道家呼吸导引节律（吸4秒→屏4秒→呼8秒→息2秒）。

C# WinForms + GDI+ 自绘，零外部依赖，~40KB exe / ~14MB 运行时。当前 v3.0，已开源（MIT），已发 CSDN 文章。
