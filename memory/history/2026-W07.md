# QuickPaths 滚轮缩放功能



## Context



用户希望悬浮圆点支持滚轮缩放（拖大/缩小）。当前圆点固定 78×32 像素，`ResizeMode = 'NoResize'`。WPF 矢量渲染，放大不会模糊。



## 方案：LayoutTransform + MouseWheel



用 WPF 的 `LayoutTransform`（`ScaleTransform`）实现整体缩放。**不需要修改动画/波形代码**——LayoutTransform 在布局层面缩放，内部所有坐标自动跟着缩放。



### 修改文件



仅改 `QuickPaths.ps1`，涉及 4 处：



### 1. 初始化 ScaleTransform（~Line 305，dot 创建后）



```powershell

$script:dotScale = New-Object System.Windows.Media.ScaleTransform(1.0, 1.0)

$script:dot.LayoutTransform = $script:dotScale

```



### 2. 添加 MouseWheel 事件（~Line 735，现有鼠标事件后面）



```powershell

$script:dot.Add_MouseWheel({

    param($s, $e)

    $delta = if ($e.Delta -gt 0) { 0.1 } else { -0.1 }

    $newScale = [Math]::Round($script:dotScale.ScaleX + $delta, 1)

    $newScale = [Math]::Max(0.5, [Math]::Min(3.0, $newScale))

    $script:dotScale.ScaleX = $newScale

    $script:dotScale.ScaleY = $newScale

    SaveConfig

})

```



- 范围：0.5x ~ 3.0x（默认 1.0x）

- 步进：0.1

- 滚一下立即生效 + 持久化



### 3. SaveConfig 加 scale 字段（~Line 228）



```powershell

$json = @{ left = $l; top = $t; claudeMode = $script:claudeMode; scale = $script:dotScale.ScaleX } | ConvertTo-Json

```



### 4. LoadConfig 读取 scale 并应用（~Line 220，return 前）



```powershell

if ($pos.PSObject.Properties['scale']) {

} else {

    $script:initialScale = 1.0

}
```

在 dot 创建后用 `$script:initialScale` 初始化 ScaleTransform。

`LayoutTransform` 是 WPF 布局系统的一部分。呼吸动画 timer 继续在原始 78×32 坐标系里绘制 Polyline 点，WPF 渲染时自动把整个 dot Border（含 Canvas、Polyline、Ellipse）按 ScaleTransform 缩放绘制。矢量渲染，无损清晰。

dot 和 panel 是 Grid 的兄弟节点。LayoutTransform 只作用于 dot，panel（路径列表）大小不变。
