# QuickPaths v3.0 — WPF → WinForms 轻量化重写



## Context



QuickPaths 当前基于 WPF，内存占用 ~307 MB（WorkingSet），对于一个桌面悬浮小工具来说过重。目标：用 WinForms 重写，内存降至 30-50 MB，同时保持所有功能和稳定性。WPF 版本已备份到 `_archive/v2.0-wpf/`。



## 约束



- 仍用 csc.exe 编译，C# 5 语法，零外部依赖

- 单文件 `QuickPaths.cs`

- 保持三模式运行（正常 / --install / --uninstall）

- 保持三层保活机制

- 保持 paths.json / config.json 兼容（用户升级无缝）



## 核心改动



### 1. 窗口基础 → Form 替代 Window



| WPF | WinForms |

|-----|---------|

| `Window { AllowsTransparency, WindowStyle=None }` | `Form { FormBorderStyle=None, TopMost=true }` |

| `SizeToContent` | 手动计算 `Size` |

| `Dispatcher.BeginInvoke` | `Control.BeginInvoke` |

| `SystemParameters.WorkArea` | `Screen.PrimaryScreen.WorkingArea` + 多屏遍历 |



**透明 + 圆角方案**：

- **圆点窗口**：使用 `SetLayeredWindowAttributes` 配合 `TransparencyKey` 实现背景透明，`Region = new Region(GraphicsPath)` 实现圆角裁剪

- 或者更简单：不做全窗口透明，dot 本身是小方块带圆角绘制，背景色 = `TransparencyKey` 即可



### 2. 动画 → Timer + GDI+ 自绘



| WPF | WinForms |

|-----|---------|

| `DispatcherTimer` (50ms) | `System.Windows.Forms.Timer` (50ms) |

| `Polyline.Points` | `Graphics.DrawLines(pen, points)` |

| `Ellipse` + `RadialGradientBrush` | `Graphics.FillEllipse` + `PathGradientBrush` |

| `DropShadowEffect` | 省略或用 `CS_DROPSHADOW` 系统级阴影 |

| `ScaleTransform` | 手动缩放坐标 + `Form.Size` |



**呼吸动画保持不变**：正弦波计算逻辑完全复用，只是渲染方式从 WPF 控件树改为 `OnPaint` + GDI+。



### 3. 面板 → OwnerDraw Panel



| WPF | WinForms |

|-----|---------|
